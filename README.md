# Мир Кирпи и Гофера 🦔🐹

Детский сайт с добрыми историями, аудиосказками, играми и раскрасками.
Превью-версия по макетам.

## Страницы

- `index.html` — главная (hero, разделы, «Наши друзья», «Новые приключения»)
- `characters.html` — персонажи: Друзья Кирпи и Гофера
- `styles.css` — оформление (тёплая сказочная тема)
- `assets/` — круглые аватары персонажей (заглушки-SVG)

## Как посмотреть

Открой `index.html` в браузере двойным кликом, либо запусти локальный сервер:

```bash
python3 -m http.server 8000
```

Затем открой <http://localhost:8000>.

## Заменить иллюстрации на свои

Сейчас вместо рисунков стоят эмодзи и цветные заглушки в `assets/`.
Положи свои картинки (фоны hero и портреты героев) и поменяй пути
в `index.html` / `characters.html`.
