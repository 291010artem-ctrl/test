# Моё фото — превью сайта

Личное фото-портфолио. Пока это **превью только для меня**, позже выпустим для всех.

## Как посмотреть

Просто открой `index.html` в браузере (двойной клик).

Либо запусти локальный сервер, чтобы всё работало как «по-настоящему»:

```bash
python3 -m http.server 8000
```

Затем открой <http://localhost:8000>.

## Как вставить свои фото

1. Положи свои снимки в папку `images/` (например `images/my-1.jpg`).
2. Открой `script.js` и поменяй пути в списке `photos`:

   ```js
   const photos = [
     { src: "images/my-1.jpg", title: "Название" },
     // ...
   ];
   ```

Сейчас вместо фото стоят цветные плейсхолдеры — их можно удалить.

## Файлы

- `index.html` — структура страницы
- `styles.css` — оформление (тёмная тема, акцентный цвет)
- `script.js` — галерея и просмотр фото (лайтбокс)
- `images/` — картинки
